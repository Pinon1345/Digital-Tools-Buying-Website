Project Name: DIGITAL TOOLS BUYING WEBSITE
Project Maker: FOURKAN BIN ILIAS

* Project Overview: 

This project is a modern web application where any users can easily explore, compare and willing to purchase premium digital tools such as AI writing tools, Design resources, SEO tools and Productivity software. This project is fully responsive for all small/medium/large devices. Also maintain valuable users friendly UI. The platform provides a clean user interface and smooth user experience to help users easily find and buy tools based on their needs.

* Objective:

a. To build a user-friendly platform for buying digital tools
b. To implement dynamic dummy data rendering using JSON
c. To create an interactive UI with filtering and cart system

* Technologies:

1. Vite
2. React.js
3. Tailwind.css
4. DaisyUI
5. React.icons
6. React toastify (NPM package)

* Three Features: 

~ Modern Responsive UI
~ Add to Cart Functionality
~ Filter and Tabs System
